# 🎬 WebFlyx

A practice repository built while completing the **Learn Git** course on [Boot.dev](https://boot.dev).

## 📚 About

WebFlyx is a fictional streaming platform used as the project context for learning Git fundamentals. This repo tracks the progress and exercises completed throughout the course.

## 🛠️ What I Learned

- Initializing and cloning repositories
- Staging, committing, and viewing history
- Working with branches and merging
- Resolving merge conflicts
- Using `.gitignore` to exclude files
- Pushing and pulling from remote repositories

## 📁 Structure

| File/Folder | Description |
|---|---|
| `classics.csv` | List of classic titles in the WebFlyx library |
| `contents.md` | Content catalogue |
| `titles.md` | Title listings |
| `dune.md` | Entry for the movie Dune |
| `quotes/` | Collection of quotes |
| `.gitignore` | Files excluded from version control |
